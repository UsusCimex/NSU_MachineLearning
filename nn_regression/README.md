# Лабораторная 12 — Нейронная сеть прямого распространения (регрессия)

## Теоретические основы

**Нейронная сеть прямого распространения (Feedforward Neural Network)** для задачи регрессии представляет собой параметрическую модель, состоящую из последовательности слоёв нейронов. Сигнал проходит от входного слоя через один или несколько скрытых слоёв к выходному без обратных связей и циклов. Каждый нейрон выполняет линейное преобразование входов с последующим применением нелинейной функции активации.

Архитектура сети для регрессии практически идентична архитектуре для классификации. Принципиальное отличие сосредоточено на **выходном слое**: вместо функции softmax или sigmoid, преобразующей выходы в вероятности классов, используется **линейная активация** (то есть активация отсутствует). Выходной нейрон непосредственно возвращает вещественное число — предсказание непрерывной целевой переменной. Соответственно, изменяется и функция потерь: вместо перекрёстной энтропии применяется среднеквадратичная ошибка или её аналоги.

### Когда нейронная сеть предпочтительнее простых методов

Нейронная сеть для регрессии целесообразна в ситуациях, когда зависимость между признаками и целевой переменной существенно **нелинейна** и не поддаётся адекватному описанию линейной моделью или полиномиальной регрессией. Линейная регрессия, Ridge, Lasso подбирают гиперплоскость; деревья решений строят кусочно-постоянную аппроксимацию. Нейронная сеть, благодаря нелинейным активациям и многослойной композиции, способна аппроксимировать произвольные непрерывные функции (универсальная теорема аппроксимации).

### Интуитивный пример

Пусть требуется предсказать потребление электроэнергии зданием по набору признаков: температура воздуха, влажность, день недели, время суток, площадь помещений. Зависимость носит сложный нелинейный характер с взаимодействиями признаков (например, влияние температуры зависит от времени суток). Линейная регрессия не способна уловить подобные зависимости без ручного конструирования признаков. Нейронная сеть автоматически обнаруживает нелинейные закономерности и взаимодействия, формируя в скрытых слоях информативные внутренние представления данных.

## Математический аппарат

### Архитектура сети

Пусть входной вектор $\mathbf{x} \in \mathbb{R}^n$ содержит $n$ признаков. Сеть состоит из $L$ слоёв. Для $l$-го слоя обозначим:

- $\mathbf{W}^{(l)}$ — матрица весов размера $(d_{l} \times d_{l-1})$,
- $\mathbf{b}^{(l)}$ — вектор смещений размера $d_{l}$,
- $\sigma^{(l)}$ — функция активации $l$-го слоя.

Прямое распространение сигнала описывается рекуррентным соотношением:

$$\mathbf{z}^{(l)} = \mathbf{W}^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}, \quad \mathbf{a}^{(l)} = \sigma^{(l)}(\mathbf{z}^{(l)})$$

где $\mathbf{a}^{(0)} = \mathbf{x}$ — входной вектор.

| Слой | Размерность | Функция активации | Назначение |
|---|---|---|---|
| **Входной** | $d_0 = n$ (число признаков) | — | Приём входных данных |
| **Скрытый 1** | $d_1$ (гиперпараметр) | ReLU, tanh, Leaky ReLU | Извлечение нелинейных признаков |
| **Скрытый 2** | $d_2$ (гиперпараметр) | ReLU, tanh, Leaky ReLU | Формирование представлений более высокого уровня |
| $\ldots$ | $\ldots$ | $\ldots$ | $\ldots$ |
| **Выходной** | $d_L = 1$ | **Линейная** (тождественная) | Предсказание $\hat{y}$ |

Выход сети для задачи регрессии:

$$\hat{y} = \mathbf{w}^{T} \mathbf{a}^{(L-1)} + b$$

где $\mathbf{w} \in \mathbb{R}^{d_{L-1}}$ — веса последнего слоя, $b$ — смещение. Функция активации на выходном слое отсутствует (применяется тождественное отображение), что позволяет модели предсказывать произвольные вещественные значения.

### Функции активации скрытых слоёв

| Функция | Формула | Область значений | Особенности |
|---|---|---|---|
| **ReLU** | $\sigma(z) = \max(0, z)$ | $[0, +\infty)$ | Быстрая сходимость; проблема «мёртвых нейронов» при $z < 0$ |
| **Leaky ReLU** | $\sigma(z) = \max(\alpha z, z)$, $\alpha \ll 1$ | $(-\infty, +\infty)$ | Устраняет проблему мёртвых нейронов |
| **tanh** | $\sigma(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}$ | $(-1, 1)$ | Выход центрирован относительно нуля; проблема затухающих градиентов |
| **Sigmoid** | $\sigma(z) = \frac{1}{1 + e^{-z}}$ | $(0, 1)$ | Используется редко в скрытых слоях из-за затухания градиентов |

### Функции потерь

#### Среднеквадратичная ошибка (MSE)

$$L_{MSE} = \frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2$$

Наиболее распространённая функция потерь для регрессии. Дифференцируема всюду, что обеспечивает гладкий ландшафт оптимизации. Штрафует крупные ошибки квадратично, поэтому **чувствительна к выбросам**.

#### Средняя абсолютная ошибка (MAE)

$$L_{MAE} = \frac{1}{m}\sum_{i=1}^{m}|y_i - \hat{y}_i|$$

Более устойчива к выбросам, чем MSE, поскольку штраф линеен. Недостаток — недифференцируемость в точке $y_i = \hat{y}_i$, что может замедлять оптимизацию вблизи оптимума.

#### Функция потерь Хьюбера (Huber Loss)

$$L_\delta(y, \hat{y}) = \begin{cases} \frac{1}{2}(y - \hat{y})^2 & \text{if } |y - \hat{y}| \leq \delta \\ \delta|y - \hat{y}| - \frac{1}{2}\delta^2 & \text{otherwise} \end{cases}$$

Комбинирует достоинства MSE и MAE: при малых ошибках ($|y - \hat{y}| \leq \delta$) ведёт себя как MSE (гладкая, дифференцируемая), при больших — как MAE (линейный рост, устойчивость к выбросам). Параметр $\delta$ определяет порог перехода.

| Функция потерь | Устойчивость к выбросам | Гладкость | Скорость сходимости |
|---|---|---|---|
| **MSE** | Низкая | Гладкая всюду | Высокая |
| **MAE** | Высокая | Не дифференцируема в 0 | Средняя |
| **Huber** | Высокая (при правильном $\delta$) | Гладкая всюду | Высокая |

### Метрики качества

| Метрика | Формула | Интерпретация |
|---|---|---|
| **MSE** | $\frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2$ | Средний квадрат отклонения. Чувствительна к выбросам. |
| **RMSE** | $\sqrt{\frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}$ | Корень из MSE; имеет ту же размерность, что и $y$. |
| **MAE** | $\frac{1}{m}\sum_{i=1}^{m}\|y_i - \hat{y}_i\|$ | Средний модуль отклонения. Более интерпретируема, устойчива к выбросам. |
| **$R^2$** | $1 - \frac{\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{m}(y_i - \bar{y})^2}$ | Доля объяснённой дисперсии. |

Коэффициент детерминации $R^2$ определяется следующим образом:

$$R^2 = 1 - \frac{SS_{res}}{SS_{tot}} = 1 - \frac{\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{m}(y_i - \bar{y})^2}$$

| Значение $R^2$ | Интерпретация |
|---|---|
| $R^2 = 1$ | Модель идеально описывает данные |
| $0.7 \leq R^2 < 1$ | Модель объясняет значительную долю дисперсии |
| $0 \leq R^2 < 0.7$ | Модель объясняет лишь часть дисперсии; возможно, требуется усложнение |
| $R^2 < 0$ | Модель предсказывает хуже, чем константа $\bar{y}$ |

### Обратное распространение ошибки и оптимизация

Обучение сети выполняется методом **обратного распространения ошибки (backpropagation)** в сочетании с алгоритмом оптимизации. Алгоритм backpropagation вычисляет градиенты функции потерь по всем параметрам сети с помощью цепного правила дифференцирования.

Для выходного слоя с линейной активацией и функцией потерь MSE градиент принимает простой вид:

$$\frac{\partial L}{\partial \hat{y}_i} = \frac{2}{m}(\hat{y}_i - y_i)$$

Сравним с классификацией, где градиент Cross-Entropy через softmax имеет более сложную структуру. В регрессии градиент выходного слоя — это просто масштабированная разность между предсказанием и истинным значением.

Правило обновления весов на шаге $t$:

$$\mathbf{W}^{(l)}_{t+1} = \mathbf{W}^{(l)}_t - \alpha \frac{\partial L}{\partial \mathbf{W}^{(l)}}$$

Основные оптимизаторы:

| Оптимизатор | Формула обновления | Особенности |
|---|---|---|
| **SGD** | $w_{t+1} = w_t - \alpha g_t$ | Простота; зашумлённая траектория; медленная сходимость |
| **SGD + Momentum** | $v_{t+1} = \beta v_t + g_t$; $w_{t+1} = w_t - \alpha v_{t+1}$ | Ускоренная сходимость; сглаживание осцилляций |
| **Adam** | Адаптивные learning rates на основе первого и второго моментов градиента | Хорошо работает «из коробки»; наиболее популярен на практике |

где $g_t = \frac{\partial L}{\partial w}\bigg|_{t}$ — градиент на шаге $t$, $\alpha$ — скорость обучения, $\beta$ — коэффициент момента.

## Отличия регрессионной сети от классификационной

| Характеристика | Классификация | Регрессия |
|---|---|---|
| **Активация выходного слоя** | Softmax (многоклассовая) / Sigmoid (бинарная) | Линейная (тождественная) |
| **Число выходных нейронов** | Равно числу классов (или 1 для бинарной) | 1 (для скалярной регрессии) |
| **Функция потерь** | Cross-Entropy Loss | MSE / MAE / Huber Loss |
| **Метрики** | Accuracy, Precision, Recall, F1 | RMSE, MAE, $R^2$ |
| **Диапазон выхода** | $[0, 1]$ (вероятности) | $(-\infty, +\infty)$ |
| **Нормализация выхода** | Вероятности суммируются до 1 | Часто нормализуют целевую переменную для стабильности обучения |
| **Градиент выходного слоя** | $\hat{y}_i - y_i$ (для softmax + CE) | $\frac{2}{m}(\hat{y}_i - y_i)$ (для MSE) |

## Нормализация целевой переменной

В задаче регрессии нормализация целевой переменной $y$ играет важную роль для стабильности и скорости обучения. Если значения $y$ имеют большой масштаб (например, цены в миллионах), то:

1. **Большие значения потерь** приводят к большим градиентам, что вызывает нестабильность обучения и необходимость подбора крайне малой скорости обучения.
2. **Дисбаланс масштабов** между входными признаками и целевой переменной затрудняет оптимизацию.
3. **Инициализация весов** выходного слоя рассчитана на значения порядка единицы; крупные целевые значения требуют адаптации.

Стандартный подход — **стандартизация** целевой переменной перед обучением:

$$y_{norm} = \frac{y - \mu_y}{\sigma_y}$$

где $\mu_y$ и $\sigma_y$ — среднее и стандартное отклонение $y$ на обучающей выборке. После получения предсказания выполняется обратное преобразование:

$$\hat{y} = \hat{y}_{norm} \cdot \sigma_y + \mu_y$$

Аналогично нормализуются и входные признаки с помощью `StandardScaler`.

## Регуляризация

Нейронные сети обладают высокой ёмкостью (числом параметров), что делает их склонными к переобучению. Методы регуляризации ограничивают сложность модели.

| Метод | Описание | Реализация в PyTorch |
|---|---|---|
| **Early stopping** | Обучение прекращается, когда ошибка на валидационной выборке перестаёт уменьшаться в течение $p$ эпох (patience). | Ручная реализация через отслеживание `val_loss` |
| **Dropout** | На каждом шаге обучения случайная доля нейронов скрытого слоя отключается с вероятностью $p$. Снижает коадаптацию нейронов. | `nn.Dropout(p)` |
| **Weight decay (L2)** | К функции потерь добавляется штраф $\lambda \sum w^2$. Ограничивает величину весов. | Параметр `weight_decay` в оптимизаторе |
| **Batch Normalization** | Нормализация активаций внутри мини-батча. Ускоряет обучение и действует как регуляризатор. | `nn.BatchNorm1d(d)` |

### Early stopping

Алгоритм ранней остановки:
1. Разделить данные на обучающую и валидационную выборки.
2. После каждой эпохи вычислить ошибку на валидационной выборке.
3. Если ошибка не уменьшилась в течение $p$ последовательных эпох, остановить обучение.
4. Восстановить веса модели с эпохи, на которой валидационная ошибка была минимальной.

### Dropout

При обучении каждый нейрон с вероятностью $p$ исключается из вычислений. Это эквивалентно обучению ансамбля $2^d$ подсетей (где $d$ — число нейронов с dropout). При инференсе dropout не применяется, а выходы масштабируются на $(1 - p)$.

## Гиперпараметры

| Гиперпараметр | Типичные значения | Описание |
|---|---|---|
| `learning_rate` ($\alpha$) | $10^{-4}$ — $10^{-2}$ | Скорость обучения. Слишком большое значение вызывает расходимость, слишком малое — медленную сходимость. |
| `hidden_layers` | 1–5 слоёв | Число скрытых слоёв. Более глубокие сети способны моделировать более сложные зависимости, но сложнее обучаются. |
| `hidden_dim` | 32–512 нейронов на слой | Число нейронов в каждом скрытом слое. Определяет ёмкость модели. |
| `activation` | ReLU, Leaky ReLU, tanh | Функция активации скрытых слоёв. ReLU — стандартный выбор. |
| `optimizer` | Adam, SGD, RMSprop | Алгоритм оптимизации. Adam рекомендуется как базовый вариант. |
| `batch_size` | 16–256 | Размер мини-батча. Малые батчи дают лучшую генерализацию, большие — стабильнее градиенты. |
| `epochs` | 50–1000 | Максимальное число эпох обучения. Используется совместно с early stopping. |
| `dropout_rate` ($p$) | 0.0–0.5 | Доля отключаемых нейронов. Типичное значение — 0.1–0.3 для регрессии. |
| `weight_decay` ($\lambda$) | $10^{-5}$ — $10^{-2}$ | Коэффициент L2-регуляризации в оптимизаторе. |

## Преимущества и недостатки

### Преимущества

| Свойство | Пояснение |
|---|---|
| Универсальная аппроксимация | Согласно теореме Цыбенко, сеть с одним скрытым слоем и достаточным числом нейронов способна аппроксимировать любую непрерывную функцию на компакте с произвольной точностью. |
| Моделирование нелинейных зависимостей | Автоматическое извлечение нелинейных признаков и взаимодействий без ручного feature engineering. |
| Масштабируемость | Эффективно работает с большими объёмами данных благодаря мини-батчевой оптимизации и GPU-ускорению. |
| Гибкость архитектуры | Число слоёв и нейронов подбирается под задачу; возможно добавление специализированных слоёв (BatchNorm, Dropout). |
| Автоматическое извлечение признаков | Скрытые слои формируют иерархические представления данных, снижая необходимость в ручном конструировании признаков. |

### Недостатки

| Свойство | Пояснение |
|---|---|
| Отсутствие интерпретируемости | Веса и активации скрытых слоёв не имеют прямой содержательной интерпретации, в отличие от коэффициентов линейной регрессии. |
| Требовательность к объёму данных | Для обучения сети с большим числом параметров необходим достаточный объём данных, иначе высок риск переобучения. |
| Чувствительность к гиперпараметрам | Результат существенно зависит от архитектуры, скорости обучения, регуляризации. Требуется тщательный подбор. |
| Вычислительная сложность | Обучение требует значительных вычислительных ресурсов по сравнению с линейными моделями и деревьями решений. |
| Склонность к переобучению | Высокая ёмкость модели при недостатке данных или отсутствии регуляризации приводит к запоминанию обучающей выборки. |
| Невоспроизводимость | Результат зависит от случайной инициализации весов и порядка подачи мини-батчей. |

## Когда использовать

- Зависимость между признаками и целевой переменной **существенно нелинейна** и не описывается полиномиальными или кусочно-линейными моделями
- Имеется **большой объём данных** (тысячи и более наблюдений), достаточный для обучения сети без переобучения
- Необходимо моделировать **сложные взаимодействия** между признаками без ручного конструирования кросс-признаков
- Линейная регрессия, Ridge/Lasso и деревья решений показывают **неудовлетворительное качество** на данной задаче
- Допустима **потеря интерпретируемости** модели ради повышения точности предсказаний
- Доступны **вычислительные ресурсы** для обучения (GPU ускоряет процесс при больших данных и сложных архитектурах)
