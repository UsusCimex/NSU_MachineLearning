# Построение линии регрессии

## Теоретические основы

**Линейная регрессия** - один из фундаментальных методов машинного обучения и статистики, предназначенный для моделирования линейной зависимости между зависимой (целевой) переменной $y$ и одним или несколькими независимыми признаками $x_1, x_2, \ldots, x_n$. Метод решает задачу **регрессии**: по набору входных признаков предсказывается непрерывное числовое значение.

### Простая и множественная регрессия

| Вид регрессии | Формула | Описание |
|---|---|---|
| **Простая (парная)** | $y = w_0 + w_1 x$ | Один признак - одна зависимая переменная. Геометрически - прямая на плоскости. |
| **Множественная** | $y = w_0 + w_1 x_1 + \ldots + w_n x_n$ | Несколько признаков. Геометрически - гиперплоскость в пространстве признаков. |

### Интуитивный пример

Пусть требуется предсказать стоимость квартиры по её площади. Имеется выборка наблюдений:

| Площадь ($x$), м² | Цена ($y$), млн руб. |
|---|---|
| 30 | 3.2 |
| 45 | 4.5 |
| 60 | 5.8 |
| 80 | 7.1 |

Линейная регрессия подбирает прямую $y = w_0 + w_1 x$, которая наилучшим образом (в смысле минимизации суммарной квадратичной ошибки) проходит через облако точек. Полученные коэффициенты имеют содержательную интерпретацию: $w_1$ показывает, на сколько миллионов рублей возрастает цена при увеличении площади на 1 м², а $w_0$ - базовая стоимость при нулевой площади (формальный параметр модели).

При добавлении дополнительных признаков (этаж, район, год постройки) модель переходит к множественной регрессии и строит гиперплоскость в многомерном пространстве.

### Ключевые понятия

- **Целевая переменная (target)** - непрерывная величина, которую модель предсказывает
- **Признаки (features)** - входные переменные, на основании которых строится предсказание
- **Веса (коэффициенты) модели** $w_0, w_1, \ldots, w_n$ - параметры, определяющие вклад каждого признака
- **Свободный член (intercept)** $w_0$ - смещение гиперплоскости относительно начала координат

## Математический аппарат

### Модель

Линейная регрессия задаёт следующее отображение:

$$\hat{y} = w_0 + w_1 x_1 + w_2 x_2 + \ldots + w_n x_n$$

В матричной форме запись компактна. Пусть матрица объектов $\mathbf{X}$ размера $(m \times (n+1))$ содержит столбец единиц для свободного члена, а вектор параметров $\mathbf{w}$ имеет размер $(n+1)$. Тогда:

$$\hat{\mathbf{y}} = \mathbf{X}\mathbf{w}$$

### Функция потерь (MSE)

Качество модели оценивается **среднеквадратичной ошибкой** (Mean Squared Error):

$$MSE = \frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2$$

где $m$ - число наблюдений, $y_i$ - истинное значение, $\hat{y}_i$ - предсказание модели. В матричной форме:

$$MSE = \frac{1}{m}(\mathbf{y} - \mathbf{X}\mathbf{w})^T(\mathbf{y} - \mathbf{X}\mathbf{w})$$

Задача обучения - найти вектор весов $\mathbf{w}$, минимизирующий $MSE$.

### Аналитическое решение (нормальное уравнение)

Приравнивая градиент функции потерь к нулю, получаем замкнутое решение:

$$\mathbf{w} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}$$

Данное решение называется **методом наименьших квадратов (OLS)**. Оно применимо, когда матрица $\mathbf{X}^T\mathbf{X}$ обратима, то есть признаки линейно независимы. Вычислительная сложность составляет $O(n^3)$ по числу признаков, что делает метод непрактичным при очень большом количестве признаков.

### Градиентный спуск

При большом числе признаков или наблюдений используется итеративный метод - **градиентный спуск**. Правило обновления весов на каждом шаге:

$$w := w - \alpha \frac{\partial L}{\partial w}$$

где $\alpha$ - **скорость обучения (learning rate)**, $L$ - функция потерь. Для MSE частная производная по вектору весов:

$$\frac{\partial MSE}{\partial \mathbf{w}} = -\frac{2}{m}\mathbf{X}^T(\mathbf{y} - \mathbf{X}\mathbf{w})$$

Разновидности градиентного спуска:

| Метод | Описание |
|---|---|
| **Batch GD** | Градиент вычисляется по всей выборке. Стабильная сходимость, высокая стоимость итерации. |
| **Stochastic GD (SGD)** | Градиент вычисляется по одному случайному наблюдению. Быстрые итерации, зашумлённая траектория. |
| **Mini-batch GD** | Градиент вычисляется по подвыборке фиксированного размера. Компромисс между двумя предыдущими. |

### Коэффициент детерминации $R^2$

Метрика $R^2$ показывает, какую долю дисперсии целевой переменной объясняет модель:

$$R^2 = 1 - \frac{\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{m}(y_i - \bar{y})^2} = 1 - \frac{SS_{res}}{SS_{tot}}$$

| Значение $R^2$ | Интерпретация |
|---|---|
| $R^2 = 1$ | Модель идеально описывает данные |
| $R^2 = 0$ | Модель предсказывает не лучше, чем константа $\bar{y}$ |
| $R^2 < 0$ | Модель предсказывает хуже, чем среднее значение |

## Гиперпараметры и регуляризация

### Основные параметры модели

| Параметр | Значение по умолчанию | Описание |
|---|---|---|
| `fit_intercept` | `True` | Добавлять ли свободный член $w_0$. Если данные центрированы, можно отключить. |
| `StandardScaler` | не применяется | Стандартизация признаков ($\mu=0$, $\sigma=1$). Критически важна для градиентного спуска и регуляризованных моделей. |
| `normalize` | `False` (deprecated) | Устаревший параметр; рекомендуется использовать `sklearn.preprocessing.StandardScaler` в пайплайне. |

### Регуляризация

При большом числе признаков или мультиколлинеарности обычная линейная регрессия склонна к **переобучению**. Регуляризация добавляет штраф за величину весов к функции потерь.

| Метод | Функция потерь | Описание |
|---|---|---|
| **Ridge (L2)** | $L = MSE + \lambda \sum_{j=1}^{n} w_j^2$ | Штрафует большие веса, сжимая их к нулю, но не обнуляя. Устойчив к мультиколлинеарности. |
| **Lasso (L1)** | $L = MSE + \lambda \sum_{j=1}^{n} \|w_j\|$ | Способен обнулять веса, выполняя **отбор признаков** (feature selection). |
| **ElasticNet** | $L = MSE + \lambda_1 \sum \|w_j\| + \lambda_2 \sum w_j^2$ | Комбинация L1 и L2. Параметр `l1_ratio` регулирует соотношение между штрафами. |

Формулы регуляризации в развёрнутом виде:

**Ridge (гребневая регрессия):**

$$L_{Ridge} = \frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{n} w_j^2$$

Аналитическое решение:

$$\mathbf{w}_{Ridge} = (\mathbf{X}^T\mathbf{X} + \lambda \mathbf{I})^{-1}\mathbf{X}^T\mathbf{y}$$

**Lasso (регрессия с L1-штрафом):**

$$L_{Lasso} = \frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{n} |w_j|$$

Аналитического решения не существует; оптимизация выполняется методом координатного спуска.

| Гиперпараметр | Модель | Описание |
|---|---|---|
| `alpha` ($\lambda$) | Ridge, Lasso, ElasticNet | Сила регуляризации. Чем больше значение, тем сильнее штраф за величину весов. |
| `l1_ratio` | ElasticNet | Доля L1-штрафа в общей регуляризации ($0$ - чистый Ridge, $1$ - чистый Lasso). |

## Преимущества и недостатки

### Преимущества

| Свойство | Пояснение |
|---|---|
| Простота и интерпретируемость | Каждый коэффициент $w_j$ имеет явный смысл: вклад $j$-го признака в предсказание. |
| Аналитическое решение | Для небольших задач веса вычисляются точно через нормальное уравнение. |
| Быстрое обучение и предсказание | Вычислительная сложность предсказания - $O(n)$ на объект. |
| Хорошо изученная теория | Обширная статистическая база: доверительные интервалы, проверка значимости коэффициентов, анализ остатков. |
| Легко масштабируется | Эффективно работает с большими датасетами при использовании SGD. |

### Недостатки

| Свойство | Пояснение |
|---|---|
| Предположение линейности | Не способна моделировать нелинейные зависимости без ручного конструирования полиномиальных признаков. |
| Чувствительность к выбросам | Квадратичная функция потерь сильно реагирует на аномальные значения. |
| Мультиколлинеарность | При сильной корреляции признаков матрица $\mathbf{X}^T\mathbf{X}$ становится плохо обусловленной; веса нестабильны. |
| Не выполняет отбор признаков | Базовая модель (без регуляризации) использует все признаки, даже нерелевантные. |
| Требует масштабирования | Для корректной регуляризации и градиентного спуска признаки должны быть приведены к одному масштабу. |

---

## Когда использовать

- Зависимость между признаками и целевой переменной приблизительно **линейна**
- Требуется **интерпретируемая** модель, в которой можно объяснить вклад каждого признака
- Необходим быстрый и надёжный **baseline** перед применением более сложных алгоритмов
- Число признаков сопоставимо с числом наблюдений или превышает его - в этом случае применяется **Lasso** или **ElasticNet** для отбора признаков
- Данные содержат **мультиколлинеарные** признаки - применяется **Ridge** для стабилизации решения
- Объём данных велик и требуется масштабируемое решение - применяется **SGD-оптимизация**

## Реализация в ноутбуке

**Датасет:** [Auto MPG](https://www.kaggle.com/datasets/uciml/autompg-dataset) - характеристики автомобилей 1970-80-х годов (398 записей). Задача - предсказать расход топлива (mpg) по техническим характеристикам: цилиндры, объём двигателя, мощность, масса, разгон, год выпуска, происхождение.

**Ключевые результаты:**

| Модель | R2 | MAE | RMSE |
|---|---|---|---|
| LinearRegression (baseline) | 0.7923 | 2.4620 | 3.2561 |
| Ridge (alpha=2.68) | 0.7893 | 2.4670 | 3.2790 |

Наибольший вклад в предсказание - масса автомобиля (отрицательный) и год выпуска (положительный). Регуляризация не даёт заметного улучшения при данном соотношении признаков и наблюдений.
